# Data

Thư mục quản lý dữ liệu theo vòng đời.

```text
raw/        Raw CERT r4.2 dataset, local only, ignored by git
sample/     Sample nhỏ để smoke test/demo nhanh
interim/    Dữ liệu trung gian generated, ignored by git
processed/  Dữ liệu processed generated, ignored by git
schemas/    Schema và data dictionary, commit được
```

Raw dataset đầy đủ không nên commit vào repo.
