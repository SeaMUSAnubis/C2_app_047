# Frontend

Dashboard React/Next.js cho UEBA Endpoint Monitoring.

## Trách nhiệm

- Login và role-aware navigation.
- Overview dashboard.
- User/device/log/alert list views.
- Alert detail với timeline, risk factors và explanation.
- API client theo contract trong `docs/API_CONTRACT.md`.

## Cấu trúc dự kiến

```text
src/
  components/
  features/
    auth/
    dashboard/
    users/
    devices/
    logs/
    alerts/
  lib/api/
  lib/utils/
  styles/
  tests/
e2e/
```
